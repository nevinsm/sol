# Security Audit Synthesis

Consolidate findings from all five audit dimensions into a prioritized security report.

Scope: {{scope}}
{{focus}}

Read findings from each audit leg and produce:
- Executive summary with overall security posture assessment
- Critical findings (P0): immediate action required — active vulnerabilities, exposed secrets, broken authentication
- High-priority findings (P1): fix before next release — significant risks with clear exploit paths
- Medium findings (P2): improvement opportunities — defense-in-depth gaps, hardening recommendations
- Low findings (P3): best-practice suggestions — code hygiene, future-proofing
- Remediation priorities: ordered list of recommended actions with effort estimates
- Cross-cutting themes: patterns that appear across multiple audit dimensions
